## Data Flow

1. **ユーザー入力** → Frontend (SolidJS) がイベントをキャプチャ
2. **IPC Call** → `@tauri-apps/api` を使用してRust側に通信
3. **Rust処理** → Tauriコマンドハンドラーが処理を実行
4. **結果返却** → IPCを通じてFrontendに結果を返す
5. **UI更新** → SolidJSのリアクティブシステムがUIを更新
