## プロジェクト名
**Mu**

## 1. プロジェクト概要
### 1.1 コンセプト
「多機能で重い」現代のブラウザへのアンチテーゼとして開発する、**軽量・ミニマル・没入型**のWebブラウザ。ユーザーが「読む」「調べる」ことに集中できるよう、視覚的なノイズを極限まで排除する。

### 1.2 ターゲット
*   開発者、ライター、研究者など、Web上の情報収集や長文読解を行うユーザー。
*   既存のブラウザのメモリ消費や動作の重さに不満を持つユーザー。

### 1.3 対応プラットフォーム
*   **Desktop:** Windows, macOS, Linux (優先)
*   **Mobile:** iOS / Android (将来的な対応)

## 2. 機能要件 (Functional Requirements)

### 2.1 ブラウジングコア機能
*   **WebView:** OS標準のレンダリングエンジンを使用し、軽量動作を実現する。
*   **ナビゲーション:** 戻る、進む、リロード、ホームへの移動。
*   **コマンドパレット:** 従来のアドレスバーを排除し、ショートカットキー (`Ctrl+K` / `Cmd+K`) で起動するモーダルウィンドウにてURL入力や検索を行う。

### 2.2 UI/UX・カスタマイズ機能
*   **サイドバー設定:**
    *   **配置位置:** ユーザー設定により「左側 (Left)」または「右側 (Right)」を選択可能。
    *   **表示モード:**
        *   **常時表示 (Fixed):** 常にサイドバーを表示する。
        *   **自動隠蔽 (Auto-hide):** 通常は非表示。設定した画面端（左端または右端）にマウスカーソルが接触した際にスライドイン/オーバーレイ表示する。
*   **ワークスペーススタイル:** サイドバーにタブリスト、ブックマーク、設定アイコンを集約。

### 2.3 タブ管理機能
*   **垂直タブ (Vertical Tabs):** サイドバー内にリスト形式でタブを表示。
*   **タブ操作:** 新規作成、閉じる、切り替え、並べ替え（ドラッグ&ドロップはフェーズ2以降）。

### 2.4 その他
*   **広告ブロック:** 簡易的なドメインブロック機能を内蔵（将来的な拡張）。
*   **キーボードショートカット:** マウスを使わずとも主要な操作が可能なキーバインディング。

## 3. 非機能要件 (Non-Functional Requirements)
*   **パフォーマンス:** アプリ起動速度 1秒以内（コールドスタート）。
*   **リソース:** アイドル時のメモリ使用量を既存ブラウザ（Chrome等）比で大幅に削減。
*   **プライバシー:** ユーザーの閲覧履歴や行動データを外部サーバーへ送信しない。

## 4. 技術スタック (Tech Stack)
*   **Core Framework:** **Tauri v2** (Rust + WebView)
*   **Backend (System):** **Rust** (ウィンドウ制御、ファイルシステム、重い処理)
*   **Frontend (UI):** **React** + **TypeScript**
*   **Styling:** **Tailwind CSS**
*   **State Management:** **Simple Store (@simple-stack/store)** (設定、タブ状態の管理)
