# Goal
`docs/roadmap/implementation-notes.md` に `$ARGUMENTS` を追記する。

# Input
- `$ARGUMENTS`: 追記したい文章（複数行可）

# Instructions
- 既存の構成・文体に合わせて、自然な場所（通常は末尾）に追記する。
- `$ARGUMENTS` が空/曖昧な場合は、追記内容を確認してから進める。
- 追記以外の変更はしない（整形のついでのリファクタ等はしない）。

# Output
- どこに何を追記したかを 1-2 行で要約する。
